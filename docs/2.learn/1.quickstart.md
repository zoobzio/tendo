# Quickstart

## Requirements

- Go 1.21 or later
- For CUDA support: NVIDIA CUDA Toolkit with cuBLAS

## Installation

```bash
go get github.com/zoobzio/tendo
```

## Basic Usage

```go
package main

import (
    "context"
    "fmt"

    "github.com/zoobzio/tendo"
    "github.com/zoobzio/pipz"
)

func main() {
    ctx := context.Background()

    // Create tensors
    a := tendo.FromSlice([]float32{1, 2, 3, 4, 5, 6}, 2, 3)
    b := tendo.FromSlice([]float32{1, 1, 1, 2, 2, 2}, 2, 3)

    // Element-wise operations
    c, _ := tendo.Add(b).Run(ctx, a)
    fmt.Println("Add:", c.Shape()) // [2 3]

    // Create random tensors
    x := tendo.RandN(4, 4)
    y := tendo.RandN(4, 4)

    // Matrix multiplication
    z, _ := tendo.MatMul(y).Run(ctx, x)
    fmt.Println("MatMul:", z.Shape()) // [4 4]

    // Reductions
    sum, _ := tendo.Sum(-1).Run(ctx, z)
    fmt.Println("Sum:", sum.Shape()) // [4]

    // Activations
    activated, _ := tendo.ReLU().Run(ctx, z)
    probs, _ := tendo.Softmax(-1).Run(ctx, activated)
    fmt.Println("Softmax:", probs.Shape()) // [4 4]
}
```

## What's Happening

1. `FromSlice` creates a tensor from a Go slice with specified shape
2. `Add(b).Run(ctx, a)` performs element-wise addition (with broadcasting)
3. `RandN` creates a tensor with standard normal random values
4. `MatMul(y).Run(ctx, x)` performs matrix multiplication
5. `Sum(-1)` reduces along the last dimension
6. `ReLU()` and `Softmax(dim)` apply activation functions

## Common Constructors

```go
// From data
t := tendo.FromSlice(data, shape...)

// Filled tensors
zeros := tendo.Zeros(3, 4)
ones := tendo.Ones(2, 2)
full := tendo.Full(3.14, 5, 5)

// Random tensors
uniform := tendo.Rand(10, 10)    // [0, 1)
normal := tendo.RandN(10, 10)    // N(0, 1)

// Sequences
seq := tendo.Arange(0, 10, 1)          // [0, 1, 2, ..., 9]
lin := tendo.Linspace(0, 1, 100)       // 100 points from 0 to 1

// Identity matrix
eye := tendo.Eye(4)
```

## Device Selection

```go
// Default is CPU
t := tendo.Zeros(100, 100)

// Explicit device
gpu := tendo.ZerosOn(tendo.CUDADevice(0), 100, 100)
cpu := tendo.ZerosOn(tendo.CPUDevice(), 100, 100)

// Set default device
tendo.SetDefaultDevice(tendo.CUDADevice(0))
t := tendo.Zeros(100, 100) // Now creates on GPU

// Transfer between devices
cpuTensor, _ := tendo.ToCPU().Run(ctx, gpuTensor)
gpuTensor, _ := tendo.ToGPU(0).Run(ctx, cpuTensor)
```

## Pipeline Composition

```go
import "github.com/zoobzio/pipz"

// Create a simple forward pass
forward := pipz.Chain(
    tendo.MatMul(weights),
    tendo.Add(bias),
    tendo.ReLU(),
)

// Run the pipeline
output, err := forward.Run(ctx, input)
if err != nil {
    log.Fatal(err)
}
```
